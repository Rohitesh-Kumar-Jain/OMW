OMW Documentation on Documentation

This page provides documentation on best practices for documentation in the Open Multilingual Wordnet.

When in doubt we use the Google style guide.

Local Style Notes

{{ili_link(71369, "WordNet")}} is a proper name, referring to the Princeton Wordnet {{ili_link(71370, "wordnet")}} is a common noun, referring to any lexical database that follows the structure of the Princeton WordNet.

Useful Jinja2 macros

  • You can link a word to a concept via ili OMW in the jinja templates with a macro like 
          {% raw %}
      {{ili_link(71370, "wordnet")}}
      {% endraw %}
    the first argument is the ili-id, the second is the word you want shown.
